# 相關資源 (/zh-tw/docs/related-resource)



下面這些資源可以幫助你補齊技術背景。它們並不是提問模板，而是讓你更容易理解系統、網絡、軟件發佈與社區協作的基礎材料。

## 基礎知識 [#基礎知識]

如果你需要個人電腦、Unix 系統和網絡如何運作的基礎知識，參閱 [Unix 系統和網絡基本原理](http://en.tldp.org/HOWTO/Unix-and-Internet-Fundamentals-HOWTO/)。

閱讀這類材料的意義在於：當你能說清楚文件系統、進程、網絡連接、權限、標準輸入輸出等基本概念時，你的問題會更準確，回答者也更容易判斷你卡在哪一層。

## 發佈軟件或補丁 [#發佈軟件或補丁]

當你發佈軟件或補丁時，試著按[軟件發佈實踐](http://en.tldp.org/HOWTO/Software-Release-Practice-HOWTO/index.html)操作。

良好的發佈習慣能減少他人使用、測試和反饋時的成本。版本號、變更記錄、構建方式、依賴說明和已知問題寫得越清楚，別人越容易給出有效反饋。
