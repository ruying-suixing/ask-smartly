# 別問自己的家庭作業 (/zh-tw/docs/when-questions/donnot-ask-homework)



黑客們很擅長分辨哪些問題是家庭作業式的問題；因為我們中的大多數都曾自己解決這類問題。同樣，這些問題得由**你**來搞定，你會從中學到東西。你可以要求給點提示，但別要求得到完整的解決方案。

## 作業的目的就是讓你練習 [#作業的目的就是讓你練習]

家庭作業式問題並不限於學校作業。任何本質上要求你通過練習來掌握概念、完成推理或寫出實現的問題，都屬於這一類。別人直接給你完整答案，只會剝奪你學習的過程。

可以問：

* `我對題意的理解是否正確？`
* `我這個思路卡在某一步，是否有明顯誤區？`
* `我應該複習哪部分概念？`

不要問：

* `誰能直接給我完整代碼？`
* `把答案發我一下。`
* `明天要交，幫我做完。`

## 去更合適的新手渠道 [#去更合適的新手渠道]

如果你懷疑自己碰到了一個家庭作業式的問題，但仍然無法解決，試試在用戶群組，論壇或（最後一招）在項目的**用戶**郵件列表或論壇中提問。儘管黑客們**會**看出來，但一些有經驗的用戶也許仍會給你一些提示。

即使在那裡，也要展示你已經嘗試過什麼。請求提示和請求代做，是完全不同的兩件事。
