# 低聲下氣不能代替你的功課 (/zh-tw/docs/when-questions/humble-not-substitute-homewoek)



有些人明白他們不該粗魯或傲慢的提問並要求得到答覆，但他們選擇另一個極端 —— 低聲下氣：`我知道我只是個可悲的新手，一個失敗者，但...`。這既使人困擾，也沒有用，尤其是伴隨著與實際問題含糊不清的描述時更令人反感。

## 自貶不會讓問題更清楚 [#自貶不會讓問題更清楚]

低聲下氣看似謙虛，實際會把注意力從問題本身轉移到你的情緒上。回答者需要的是事實、背景、目標和嘗試記錄，不是你對自己的評價。

別用原始靈長類動物的把戲來浪費你我的時間。取而代之的是，儘可能清楚地描述背景條件和你的問題情況。這比低聲下氣更好地定位了你的位置。

更好的表達是：`我是第一次使用這個庫，已經讀過入門文檔和 FAQ，但仍不理解下面這個錯誤。` 這既說明了你的經驗水平，也提供了有效背景。

## 新手也要認真提問 [#新手也要認真提問]

有時網頁論壇會設有專為新手提問的版面，如果你真的認為遇到了初學者的問題，到那去就是了，但一樣別那麼低聲下氣。

新手身份不是問題，拒絕做功課才是問題。坦誠說明自己在哪裡不懂，然後給出你已經嘗試過的東西，比反覆道歉更有用。
